# 📊 Bank Loan Analytics Dashboard

An interactive Excel dashboard project focused on analyzing lending performance, repayment behavior, portfolio quality, and borrower segmentation using KPI-driven reporting.

This was one of my earliest dashboard projects and helped me understand how raw financial data can be transformed into structured business intelligence reporting inside Excel.

## What the Dashboard Analyzes

| Area | Focus |
|---|---|
| Loan Applications | Lending demand analysis |
| Funded Amount | Total disbursed loans |
| Amount Received | Repayment tracking |
| Interest Rate | Lending performance |
| DTI Ratio | Borrower financial health |
| Good vs Bad Loans | Portfolio quality analysis |
| Regional Analysis | State-wise loan distribution |

## Dashboard Workflow

